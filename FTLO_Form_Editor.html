Getting Started
Edit your FTLO registration form from any device — no code required.

How to use this editor

  1. Click Load Form in the top bar and select your index.html
  2. Use the left menu — starred sections are the ones you update most often
  3. Make your edits. The preview shows you exactly what registrants see
  4. Click Apply Changes then Download
  5. Upload the downloaded index.html to GitHub — live in 1–2 minutes

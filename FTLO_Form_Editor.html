Getting Started
Edit every text field in your FTLO registration form — no code required. Access this from any device via GitHub.

How to use

  1. Click Load Form in the top bar and select your index.html
  2. Use the left menu to navigate to any section
  3. Make your edits — every field updates automatically
  4. Click Apply then Download to save a new index.html
  5. Upload to GitHub — live in 1–2 minutes
